
## Requirements
To get started, ensure the following dependencies are installed:

- **Mediapipe**: `0.10.14`
- **OpenCV (cv2)**: `4.10.0.84`
- **fastapi**: `0.115.5`
- **numpy**: `1.26.4`
- **uvicorn**: `0.32.1`

Install them using pip:

```bash
pip install mediapipe==0.10.14 opencv-python==4.10.0.84 fastapi==0.115.5 numpy==1.26.4 uvicorn==0.32.1
```

Access the OpenAPI specification at the following link:

```bash
http://localhost:8000/docs
```
